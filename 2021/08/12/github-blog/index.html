기본 세팅

  1. github에서 새 repo 만들기
    • repo 이름은 [username].github.io
    • 옵션 체크 하지말고 그대로 생성
  2. node.js 설치
  3. hexo 설치
  4. dev 폴더 안에 hexo 폴더 생성
  5. 만든 폴더 안에서 재설치
  6. ‘Next’ 테마 다운
  7. _config.yml 열기
  8. 세부사항 수정
    • title: [블로그 타이틀]
    • subtitle: [부제]
    • description: [간략한 설명]
    • author: [username]
    • url: https://[username].github.io
    • theme: Next
    • deploy:
      type : git
      repo : [위에서 만든 repo 주소]
      branch : main
  9. deployer-git 설치
